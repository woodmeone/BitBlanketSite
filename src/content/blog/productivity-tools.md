## 开发工具

### 1. VS Code

微软出品的免费代码编辑器，插件生态丰富，是开发者的首选工具。

**推荐插件：**
- GitHub Copilot - AI代码补全
- Prettier - 代码格式化
- ESLint - 代码检查

### 2. Cursor

基于VS Code的AI编辑器，内置强大的AI对话功能。

### 3. Raycast

macOS上的效率启动器，支持各种扩展和脚本。

## 设计工具

### 4. Figma

在线协作设计工具，支持实时多人编辑。

### 5. Excalidraw

手绘风格的在线白板工具，适合快速原型设计。

## 协作工具

### 6. Notion

全能笔记和知识管理工具，支持数据库、看板等多种视图。

### 7. Linear

现代化的项目管理工具，界面简洁，操作流畅。

## 效率工具

### 8. 1Password

密码管理工具，支持多平台同步。

### 9. CleanShot X

macOS上最好用的截图工具，支持录屏和标注。

### 10. Rectangle

macOS窗口管理工具，支持键盘快捷键调整窗口大小和位置。

## 总结

选择合适的工具可以大大提升工作效率。建议根据自己的需求选择2-3个核心工具深入学习，而不是追求数量。
